Instructions:
  1. Make sure you updated the Google Apps Script doGet function

Debugging Steps:

  1. Open browser console (F12) to see detailed logs
  2. Check your Google Sheet after clicking Submit
  3. Verify Apps Script:
    • Open your Google Sheet
    • Extensions → Apps Script
    • Check if doGet function handles e.parameter.name and e.parameter.email
    • View → Executions to see if requests are coming through
